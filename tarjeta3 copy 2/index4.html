    - Resumen -

    Visual Studio es un entorno de desarrollo integrado (IDE) creado por Microsoft. Está diseñado para desarrollar aplicaciones de software, sitios web, servicios web y aplicaciones móviles. Visual Studio admite múltiples lenguajes de programación, como C#, C++, Visual Basic, Python y JavaScript, lo que lo convierte en una herramienta versátil para desarrolladores. El IDE ofrece características como depuración avanzada, edición de código, control de versiones, pruebas automáticas, y una amplia gama de extensiones para personalizar la funcionalidad. Es utilizado tanto por desarrolladores principiantes como por profesionales para crear aplicaciones de escritorio, web y móviles en diversas plataformas.

    Simulación

    Para poder crear una pagina web en visual studio, debemos de crear una carpeta nueva en la cual iran distintas sub carpetas las cuales derena tener todo lo que quieras que tenga tu pagina web, videos, imagenes, etc. Una vez ya tengamos uestra carpeta lista ya podremos abrir nuestra carpeta en el programa y asi poder ir creando la pagina web Construcción Ya que tenemos la carpeta nueva lista, entraremos a nuestro programa de visial studio y una vez dentro, dremos clic en File y se abrira una lista con las distintas opiones que puedes realizar ahí, daremos clic en la de new file y nombramos nuestro proyecto como queramos y una ez creado, tendremos una barra a la izquierda de la pantalla, ahí abriremos nuestra nueva carpeta y ahora si empezaremos a programar nuestra pagina web en ella.
