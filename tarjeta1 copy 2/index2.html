    Practicas brazo (con Bluetooth)

    Raul Alejandro Sanchez Lora, Montserrat Vazquez Guzman
    Ing Mecatronica
    Fecha: Mes dia, Año
    Arduino Electrónica

    - Resumen -

    Un brazo robótico es un dispositivo mecánico creado para imitar los movimientos del brazo humano, ofreciendo una solución funcional y adaptable para diversas aplicaciones. En este caso, el modelo incluye un sistema de control por Bluetooth, que permite operarlo de forma inalámbrica mediante un teléfono móvil o una computadora. Esta característica no solo aporta comodidad, sino que también amplía las posibilidades de uso al eliminar la necesidad de conexiones físicas directas.

    - Materiales -

    --Arduino

    --jumpers

    --4 servomotores

    --4 velocimetros

    --modelo de brazo

    --HC-C5 Bluetooth

    Simulación

    En el proyecto del brazo robótico, se integraron botones adicionales a una aplicación creada con App Inventor, lo que amplió las funcionalidades y mejoró la experiencia del usuario. Esta aplicación permite controlar el brazo robótico de manera inalámbrica mediante Bluetooth, y la adición de los botones facilita realizar acciones específicas con mayor comodidad y precisión.

    Gracias a estas mejoras, la aplicación ofrece un control más detallado del brazo robótico, permitiendo a los usuarios interactuar de manera más directa y eficiente con el dispositivo. Esta funcionalidad adicional es especialmente útil en entornos educativos y recreativos, donde la personalización y la facilidad de uso son clave para el aprendizaje y la experimentación.

    - Resultados -

    El resultado fue realmente positivo al agregarle la tecnología Bluetooth. Fue un proceso sencillo y directo que permitió una mejor conectividad y funcionalidad. Al incorporar el Bluetooth, no solo mejoró la facilidad de uso, sino también la capacidad de interacción con otros dispositivos, lo que llevó a una experiencia mucho más completa. El éxito de este ajuste fue notable y mostró que la integración de nuevas tecnologías puede ser una solución eficaz y de bajo costo para mejorar el desempeño de un proyecto.
