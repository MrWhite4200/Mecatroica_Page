    Arduino Electrónica

    - Resumen -

    En este proyecto, MIT App Inventor se utilizó para diseñar una interfaz gráfica que permite al usuario encender y apagar el LED con solo presionar botones en su dispositivo móvil. La aplicación incluye elementos clave como botones, etiquetas y un componente Bluetooth Client, que facilita la comunicación entre el dispositivo móvil y el módulo Bluetooth HC-05 o HC-06 conectado al Arduino.

    En conclusión, se logró desarrollar exitosamente un sistema inalámbrico funcional utilizando una aplicación móvil, un Arduino y un módulo Bluetooth HC05. Este proyecto resalta la importancia de diseñar simultáneamente tanto la aplicación como el código de Arduino, garantizando así una comunicación coherente y precisa entre ambos, optimizando el procesamiento de los datos enviados.
