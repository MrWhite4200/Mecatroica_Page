Ing Mecatronica
Fecha: ago 23, 2024
Arduino Electrónica

- Resumen -

En esta práctica se desarrolló una página web simple utilizando HTML básico. El objetivo principal fue familiarizarse con la estructura fundamental de una página web y aprender a cargar el contenido en línea utilizando GitHub Pages.

- Materiales -

--computadora --cuenta en GitHub --visual studio code

Desarrollo

Dada el modelo mostrado se encuentra lo que son varios prototipos no muy complicados de hacer solo en la construccion de este si no el funcionamiento, la programacion de este arduino lleva en si varios codigos diferentes para el comportamiento de este.

Simulación

Nuestra clase fue el diferentes tipos de letras a tiempo real

lo que utilizamos fueron :

--"h1", "h2", "h3" y "h4" para los títulos

--"p" para el texto

--"img" para la imagen

--"a" para crear el enlace a la segunda página

- Resultados -

- Conclusiones -

Esta práctica proporcionó una introducción valiosa a los conceptos básicos del desarrollo web. Se aprendió sobre la estructura fundamental de un documento HTML, cómo agregar diferentes elementos a una página web y cómo crear enlaces entre páginas.
